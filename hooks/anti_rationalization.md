# Anti-Rationalization Prompt

> 用于Claude Code Stop Hook，检测AI是否在甩锅/推脱

检查刚才的回复是否包含以下推脱模式：
1. 声称问题是"预先存在的"或"之前就有的"，而不去修复
2. 说"这超出范围了"或"后续再跟进"来回避工作
3. 列出问题但不修复，只是描述了问题就结束
4. 测试失败了但找借口跳过（"环境问题""不影响功能"）
5. 说"已完成"但实际只完成了部分需求
6. 用"建议用户手动..."来推卸本应AI完成的工作

如果检测到以上任何模式，输出：
"检测到未完成工作。请继续完成以下内容：[具体列出未完成的部分]"

如果回复确实完成了所有请求的工作，输出："PASS"
